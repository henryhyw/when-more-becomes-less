## Paper at a glance

Thirteen open-access checkpoints (2 MLMs + 11 CLMs, 0.5B–14B) are evaluated with a **two-probe diagnostic** that contrasts a displaced repetition probe (F0–F4) against an adjacent baseline. The diagnostic isolates the contribution of *repeated lexical exposure* per se from local-recency / adjacent-token effects.

- **English panel.** Across 13 models, the displaced-probe curve falls monotonically as F grows — more becomes less. The drop is attested in every model and survives a six-condition ablation that controls for surface lexical overlap, sentence length, and item identity.
- **Mechanism (§ §5).** On four mechanism models (ModernBERT-large, Qwen2.5-1.5B, Qwen2.5-7B, OLMo-2-7B), per-target-token attention and slot-state probes localize the effect to a small mid-block budget pool whose layer-wise profile tracks a 1/N reference.
- **Cross-lingual generalization.** Four multilingual checkpoints (XLM-R-base, XLM-R-large, Qwen2.5-1.5B, Qwen2.5-7B) × four languages (Spanish, Chinese, German, French) replicate the displaced-probe decline in 14 of 15 (model, language) cells (XLM-R Chinese drops out because no single-token Chinese candidate survives the BPE filter).

## Reproduction

The pipeline has two stages. The data package already ships the outputs of Stage 1; you only need Stage 1 if you want to rerun the experiments from the original checkpoints.

### Stage 1 — data generation (GPU)

```bash
make -C code data
# or, directly:
jupyter nbconvert --to notebook --execute --inplace code/notebooks/full_paper_pipeline.ipynb
```

The notebook is restartable: each (model, panel) cell skips work whose output is already on disk. Expect ~8–12 h on a single A100. Some checkpoints are gated on Hugging Face (Gemma family); set `HF_TOKEN` first or accept the model card terms in your account.

### Stage 2 — figures (CPU, < 1 min)

```bash
make -C code figures
# or, directly:
python code/src/make_figures.py             # regenerate the four data-driven figures
python code/src/make_figures.py fig2 fig3   # subset
```

Figure 1 (`fig1_probe_schematic.pdf`) is a hand-authored static asset shipped under `data/figures/` and is **not** regenerated by the script.

### Dependencies

```bash
pip install -r code/requirements.txt
```

Python 3.11 tested; `transformers`, `accelerate`, `safetensors`, `transformer-lens`, `pandas`, `numpy`, `matplotlib`, and a small handful of helpers.

## Notes

- **Tokenization.** Single-token target/probe filtering is applied per-model (BPE / SentencePiece / WordPiece) so the final-position logit-difference readout is consistent across model families. The cross-lingual panel excludes XLM-R Chinese for exactly this reason.
- **Static figure 1.** The two-probe-design schematic is hand-authored and lives in `data/figures/fig1_probe_schematic.{pdf,svg,png}`. The script makes no attempt to regenerate it; `make clean` preserves it.
- **Mechanism subset.** The four mechanism models (ModernBERT-large, Qwen2.5-1.5B, Qwen2.5-7B, OLMo-2-7B) are wired into `code/src/make_figures.py` as `MECH_MODELS`.
